What is Antigravity?

Google Antigravity is an AI-first agentic development platform designed to revolutionize software engineering. It empowers AI agents to autonomously plan, code, test, and verify tasks across editors, terminals, and browsers. This project serves as a safe, containerized environment (sandbox) for experimenting with these powerful agentic capabilities.

About This Sandbox

The Antigravity Sandbox provides a comprehensive XFCE4 desktop environment running inside a Docker container. It offers a pristine, isolated space for developers to explore the Antigravity technology without affecting their host system.

Key Features

️

Full Desktop

Complete XFCE4 desktop experience based on Debian 12 (Bookworm) Slim.



Dual Access

Connect via standard RDP (xrdp) or directly through your web browser (NoVNC).



Persistence

Map local folders to the container to keep your work safe between sessions.

️

Pre-loaded Tools

Comes ready with Antigravity packages, Chromium, and development utilities.

Under the Hood

The sandbox uses supervisord as the process manager to orchestrate the internal environment. This ensures a robust standard init replacement within the container.

1

DBus Daemon

Critical message bus service required for XFCE4.

2

TigerVNC

Headless X11 server providing the actual desktop session.

3

NoVNC

HTML5-to-VNC bridge allowing browser-based access.

4

XRDP

Microsoft RDP server implementation for native clients.

Configuration & Security

 Non-Root User

The container runs as the user antigravity (with sudo privileges) for better security practices, rather than defaulting to root.

 Dynamic Password

Set the PASSWORD environment variable at runtime to secure your session. The entrypoint script automatically updates both system and VNC passwords.

Quick Start

Run the sandbox with the recommended configuration:

docker run -d \
  --name antigravity-sandbox \
  --cap-add=SYS_ADMIN \
  --security-opt apparmor=unconfined \
  -p 3390:3389 \
  -p 6901:6901 \
  -e PASSWORD=SecretPassword123 \
  --shm-size=1g \
  -v $(pwd)/antigravity-data:/home/antigravity \
  ghcr.io/duizendstra/antigravity-sandbox:latest

User: antigravity | Password: SecretPassword123
Web Access: http://localhost:6901 | RDP: localhost:3390
